# Word Cloud Brandwatch

Word cloud is my first project in JS. Good luck to me.

## Requirements

 * Mocha
 * Supertest
 * Node.js
 * Express
 * NPM

## Project Build

To install local dependencies:
```
$ npm install
```

To install Mocha (global)
```
$ sudo npm install mocha -g
```

Starting the service:
```
$ npm start
```

## Runnning Tests

```
$ npm test
```
